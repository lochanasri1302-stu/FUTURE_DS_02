
# 📊 Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer retention and churn patterns using **Power BI**. The dashboard provides interactive visualizations that help identify customer behavior, churn trends, and factors affecting customer retention. The insights can support businesses in making data-driven decisions to improve customer loyalty.

---

## 🎯 Objective

* Analyze customer churn and retention.
* Identify factors influencing customer churn.
* Build an interactive Power BI dashboard.
* Generate actionable insights to improve customer retention.

---

## 🛠️ Tools Used

* **Power BI**
* Microsoft Excel
* DAX (Data Analysis Expressions)

---

## 📂 Dataset

The dataset contains customer-related information such as:

* Customer ID
* Gender
* Senior Citizen
* Partner
* Dependents
* Internet Service
* Phone Service
* Multiple Lines
* Online Security
* Tenure
* Monthly Charges
* Churn Status

---

## 📈 Dashboard Features

* KPI Cards

  * Total Customers
  * Churned Customers
  * Retained Customers
  * Churn Rate
* Customer Churn Distribution (Pie Chart)
* Internet Service Analysis
* Gender-wise Churn Analysis
* Partner-wise Churn Analysis
* Senior Citizen Analysis
* Online Security Analysis
* Customer Tenure Trend
* Interactive Slicers for filtering data

---

## 🔍 Key Insights

* Churn patterns vary across different customer groups.
* Customers with shorter tenure tend to churn more frequently.
* Internet service and online security influence customer retention.
* Customer demographics also impact churn behavior.
* Interactive filters allow detailed exploration of the data.

---

## 💡 Recommendations

* Introduce loyalty programs for long-term customers.
* Improve support services for customers at higher risk of churn.
* Offer personalized plans based on customer usage patterns.
* Promote value-added services to improve customer satisfaction.
* Continuously monitor churn trends for proactive decision-making.

---

## 📌 Conclusion

This project demonstrates how **Power BI** can be used to transform customer data into meaningful insights. The dashboard helps businesses understand customer behavior and supports strategies to improve retention while reducing churn.
